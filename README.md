## 🎮 Guess My Number (Computer Guesses)  

This is a fun interactive game where **you** think of a number between 1 and 100, and the **computer** tries to guess it within **10 attempts**! 🖥️🔢  

### 🕹️ How It Works  
1. You **think of a number** between 1 and 100.  
2. The computer makes a **random guess**.  
3. You provide **feedback**:  
   - **Y** (Yes) – If the computer guessed correctly.  
   - **H** (Higher) – If your number is higher.  
   - **L** (Lower) – If your number is lower.  
4. The computer **narrows down** its guesses using logic.  
5. It has **10 attempts** to guess the correct number.  
6. If the computer **fails**, you win! 🎉  

### 🛠 Features  
✔️ Interactive and fun guessing game  
✔️ Computer uses logic to find the number  
✔️ Input validation to prevent mistakes  
✔️ Limited attempts for an extra challenge
